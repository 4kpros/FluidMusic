**Fluid music** is an Android open source audio player.

## Features
-  Play local songs : mp3, ogg, mp4, opus, etc...
-  More than 5 themes
-  Headset/Bluetooth support
-  Music filters
-  Volume controls
-  Sleep timer
-  Lyrics supports
-  Tag editor
-  Manage playlists
-  Home screen widgets
